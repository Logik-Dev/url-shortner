URL Shortner

Cahier des charges:

  1. L'utilisateur peut soumettre une URL à l'adresse [url_du_projet]/api/shorturl/new et obtient une réponse au format JSON:
    {"original url": "https://www.google.com", "short_url": "mX_EcitPv"}
  2. Si l'URL fournie est ne respecte pas le format http(s)://www.exemple.com(/routes) la réponse au format JSON sera de type:
    {"error": "invalid URL"}
    Pour être sur de la validité de l'adresse on utilisera le module node urlExists(url, cb).
  3. Lorsque l'utilisateur se rend sur la page du lien, il est redirigé vers le site original.

Création d'URL raccourcie:

POST [project_url]/api/shorturl/new - https://google.fr

Par exemple:

[project_url]/api/shorturl/6ynVpYOcQ

Redidrigera l'utilisateur vers:

https://logikdev.fr
